# Running

### 1. Clone this project

### 2. Install [yarn](https://yarnpkg.com/)

### 3. Install dependencies

```
yarn install
```

### 4. Run code

```
yarn parcel index.html
```

### 5. Go to [http://localhost:1234](http://localhost:1234)
